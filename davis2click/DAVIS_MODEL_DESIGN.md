## 1. 模型哲学

### 戴维斯双击 (Double Click)
```
股价上涨 = 盈利增长 × 估值扩张
```
- **第一腿（确定性）**: 单季盈利加速增长。这是基本面驱动，可预测性相对高。
- **第二腿（概率性）**: PE 从低位向均值回归。这是市场重定价，不可预测但可筛选有利条件。
- **核心思路**: 用第一腿筛选，用第二腿排序，用催化剂确认。

### 戴维斯双杀反转 (Double Kill Reversal)
```
当前: 股价下跌 = 盈利下滑 × 估值收缩（进行时）
反转: 盈利下滑减速 + 估值已至极值 → 双击起点
```
- **胜率更低**（左侧交易），**赔率更高**（弹簧压到极致）
- **关键难点**: 区分"暂时困境"和"永久衰退"（价值陷阱过滤器）

## 3. 因子体系

### 3.1 盈利增长因子（第一腿 — 确定性）

| 因子 | 计算方式 | 阈值 |
|------|---------|------|
| `eps_yoy` | 单季净利润同比增速 = (Q_t - Q_{t-4}) / abs(Q_{t-4}) | > 20% |
| `eps_accel` | 增速加速度 = eps_yoy_t - eps_yoy_{t-1} | > 0 |
| `revenue_yoy` | 单季营收同比增速 | > 10% |
| `gross_trend` | 毛利率连续环比改善季度数 | >= 2 |
| `cash_qual` | 经营现金流净额 / 净利润 | > 0.5 |

### 3.2 估值因子（第二腿 — 弹簧）

| 因子 | 计算方式 | 阈值 |
|------|---------|------|
| `pe_pct` | 当前 PE_TTM 在近 5 年的百分位 | < 30% |
| `pb_pct` | 当前 PB 在近 5 年的百分位 | < 30% |
| `peg` | PE_TTM / eps_yoy | < 0.8 |
| `ps_ttm` | 市销率（辅助周期股估值） | 行业相对分位 < 30% |

### 3.3 催化剂因子（确认信号）

| 因子 | 计算方式 | 阈值 |
|------|---------|------|
| `turnover_amp` | 20 日均换手率 / 60 日均换手率 | > 1.3 |
| `north_flow` | 北向资金持仓占比月度变化 | > 0 |
| `margin_trend` | 融资余额月度变化 > 0 | — |
| `holder_trend` | 股东户数季度变化 < 0 (筹码集中) | < 0 |

### 3.4 风控因子（排除层）

| 因子 | 计算方式 | 排除阈值 |
|------|---------|---------|
| `goodwill_ratio` | 商誉 / 净资产 | > 30% |
| `pledge_ratio` | 整体质押比例 | > 50% |
| `ar_ratio` | 应收账款 / 营收 | > 60% |
| `debt_ratio` | 有息负债 / 净资产 | > 200% |
| `pe_negative` | PE <= 0 | 排除 |

## 4. 筛选管道

### 4.1 双击模式 (double_click)

```
全市场 5,491 只
  ├─ 剔除 ST/*ST
  ├─ 剔除上市 < 1 年
  ├─ 剔除 PE <= 0 或 > 200
  │
  ├─ 盈利加速筛选:
  │   eps_yoy > 20% AND eps_accel > 0 AND revenue_yoy > 10%
  │
  ├─ 估值低位筛选:
  │   pe_pct < 30% AND peg < 0.8
  │
  ├─ 催化剂确认:
  │   turnover_amp > 1.3 OR north_flow > 0 OR margin_trend > 0
  │
  ├─ 风控排除:
  │   goodwill/equity < 30% AND pledge_ratio < 50%
  │
  └─ 打分排序 → TOP 20-50
```

### 4.2 双杀反转模式 (double_kill_reversal)

```
全市场 5,491 只
  ├─ 剔除 ST/*ST
  ├─ 剔除上市 < 3 年 (需要更长历史)
  ├─ 剔除 PE <= 0
  │
  ├─ 双杀确认:
  │   eps_yoy < -10% AND PE 较 1 年前收缩 > 20%
  │
  ├─ 反转信号 (至少满足 2 条):
  │   eps_accel > 0 (下滑减速)
  │   gross_trend >= 1 (毛利率企稳)
  │   north_flow > 0 OR holder_num 下降
  │
  ├─ 估值极致:
  │   pe_pct < 20% AND pb_pct < 20%
  │
  ├─ 价值陷阱排除:
  │   cash_qual > 0.3
  │   goodwill/equity < 50%
  │   pledge_ratio < 60%
  │   gross_margin 未连续 4 季下滑
  │
  └─ 打分排序 → TOP 20
```

## 5. 综合打分

### 5.1 评分公式

```
score = w1 × score_earnings + w2 × score_valuation + w3 × score_catalyst

双击模式权重:  w1=0.45,  w2=0.30,  w3=0.25
反转模式权重:  w1=0.25,  w2=0.40,  w3=0.35
```

各子项内部对原始因子做 Z-score 标准化后映射到 0-100。

### 5.2 输出字段

```
ts_code, name, industry, pe_ttm, pe_pct, pb, pb_pct,
eps_yoy, eps_accel, revenue_yoy, peg, gross_trend,
turnover_amp, north_flow_pct, pledge_ratio, cash_qual,
score, type
```

## 6. 模型测度与回测

### 6.1 因子有效性

| 指标 | 含义 | 目标 |
|------|------|------|
| IC (Rank IC) | 因子值与未来 N 日收益的 Spearman 秩相关 | abs(IC) > 0.03 |
| ICIR | IC 均值 / IC 标准差 | > 0.5 |
| 分层回测 | 按因子分 10 组，看各组平均收益单调性 | 多头组显著跑赢 |
| 因子自相关 | 因子前后期相关性 | 决定调仓频率 |

### 6.2 组合回测参数

| 参数 | 默认值 |
|------|--------|
| 回测区间 | 2015-01-01 ~ 最新 |
| 调仓频率 | 月度 |
| 持仓数量 | TOP 20-30 |
| 加权方式 | 等权 / 因子得分加权 |
| 交易成本 | 双边 0.2%（千二） |
| 基准 | 中证全指 / 沪深300 |
| 剔除 ST / 停牌 / 涨停 | 是 |

### 6.3 评价指标

- 年化收益率 / 超额收益
- 夏普比率 / 信息比率
- 最大回撤 / 最长回撤期
- 胜率（跑赢基准的月份占比）
- 月度换手率
- 容量估算（日均成交额限制）

## 9. 关键风险与限制

1. **幸存者偏差**: 回测时必须使用当时在市的股票（Tushare `stock_basic` 含退市历史）
2. **财报滞后**: 季报有 1-4 个月披露延迟，实盘需对齐信息时间
3. **小市值流动性**: 按日均成交额排名前 80% 过滤，避免滑点过大
4. **过拟合**: 阈值参数需做参数敏感性分析，避免针对历史过度优化
5. **API 限速**: income 逐只拉取约 15 分钟/轮，可通过本地缓存复用历史数据
