# 超级B1选股公式 - 伪代码描述

## 1. 基础指标计算

```
// 输入: 开盘价O, 收盘价C, 最高价H, 最低价L, 成交量V
// 历史数据引用: REF(X, n) 表示n天前的X值

FUNCTION 计算基础指标():
    趋势白线 = EMA(EMA(C, 10), 10)
    大哥黄线 = (MA(C,14) + MA(C,28) + MA(C,57) + MA(C,114)) / 4
    短期 = 100 * (C - LLV(L,3)) / (HHV(C,3) - LLV(L,3))
    长期 = 100 * (C - LLV(L,21)) / (HHV(C,21) - LLV(L,21))
    BBI = (MA(C,3) + MA(C,6) + MA(C,12) + MA(C,24)) / 4

    RETURN 趋势白线, 大哥黄线, 短期, 长期, BBI
```

## 2. 基础判定条件

```
FUNCTION 基础判定(股票代码, 历史数据):
    // 判断是否为创业板/科创板等
    IF 股票代码匹配('68', '30', '4', '8', '9') OR 过去200天存在涨幅>15%:
        振幅区间 = 8
        放宽系数 = 0.9
    ELSE:
        振幅区间 = 5
        放宽系数 = 1.0

    当日振幅 = (H - L) / L * 100
    当日涨跌幅 = ABS(C - REF(C,1)) / REF(C,1) * 100 * 放宽系数
    上涨十字星 = (C > REF(C,1)) AND (ABS(C-O)/O*100 * 放宽系数 < 1.8)

    // 形态判定
    单针下20 = (短期 <= 20 AND 长期 >= 75) OR ((长期 - 短期) >= 70)
    聚宝盆 = COUNT(长期>=75, 8天) >= 6 AND COUNT(短期<=70, 7天) >= 4 AND COUNT(短期<=50, 8天) >= 1
    双叉戟 = EVERY(长期>=75, 8天) AND COUNT(短期<=50, 6天) >= 2 AND COUNT(短期<=20, 7天) >= 1
    红肥绿瘦 = COUNT(C>=O, 15天) > 7 OR COUNT(C>REF(C,1), 11天) > 5

    RETURN 振幅区间, 当日振幅, 当日涨跌幅, 上涨十字星, 单针下20, 聚宝盆, 双叉戟, 红肥绿瘦
```

## 3. 成交量条件

```
FUNCTION 成交量判定(VOL, 历史成交量):
    VDAY = 距离过去40天最高成交量的天数

    // 大绿棒判定(40天最高量那天收阴)
    不是大绿棒 = (REF(C,VDAY) >= REF(C,VDAY+1)) OR (REF(C,VDAY) >= REF(O,VDAY))
    大绿棒 = NOT(不是大绿棒)
    大绿棒离得远 = (VDAY >= 15) AND 大绿棒

    // 缩量等级
    缩量 = (VOL < HHV(VOL,20) * 0.416) OR (VOL < HHV(VOL,50) / 3)
    回踩缩量 = (VOL < HHV(VOL,20) * 0.45) OR (VOL < HHV(VOL,50) / 3)
    适当缩量 = (VOL < HHV(VOL,20) * 0.618) OR (VOL < HHV(VOL,50) / 3)
    超缩量 = (VOL < HHV(VOL,30) / 4) OR (VOL < HHV(VOL,50) / 6)

    RETURN 不是大绿棒, 大绿棒离得远, 缩量, 回踩缩量, 适当缩量, 超缩量
```

## 4. KDJ与RSI计算

```
FUNCTION 计算KDJ_RSI(收盘价序列, 最高价序列, 最低价序列):
    // KDJ指标
    K = KDJ_K值
    J = KDJ_J值

    // RSI(3)计算
    LC = REF(C,1)
    TEMP1 = MAX(C - LC, 0)
    TEMP2 = ABS(C - LC)
    RSI = SMA(TEMP1, 3, 1) / SMA(TEMP2, 3, 1) * 100

    RETURN K, J, RSI
```

## 5. 振幅与异动判定

```
FUNCTION 异动判定(H, L, N=20, M=50):
    // 近期振幅
    LOWN = LLV(L, N)
    HIGHN = HHV(H, N)
    近期振幅 = (HIGHN - LOWN) / LOWN * 100
    近期异动 = (近期振幅 >= 15) OR ((HHV(H,12) - LLV(L,14)) / LLV(L,14) * 100 >= 11)

    // 远期振幅
    LOWM = LLV(L, M)
    HIGHM = HHV(H, M)
    远期振幅 = (HIGHM - LOWM) / LOWM * 100
    远期异动 = (远期振幅 >= 30)
    超级异动 = (近期振幅 >= 60)

    // 洗盘异动
    洗盘异动 = (COUNT(单针下20, 10天) >= 2) OR (聚宝盆) OR (双叉戟)

    RETURN 近期异动, 远期异动, 超级异动, 洗盘异动, 近期振幅, 远期振幅
```

## 6. 趋势股判定

```
FUNCTION 趋势股判定(趋势白线, 大哥黄线, C, O, BBI):
    // 做上涨趋势
    做上涨趋势 = (趋势白线 >= 大哥黄线) AND (C >= 大哥黄线 OR (C > 大哥黄线*0.975 AND C > O))

    // 强趋势股
    强趋势股 = EVERY(大哥黄线 >= REF(大哥黄线,1)*0.999, 13天) AND
               趋势白线 >= REF(趋势白线,1) AND
               EVERY(趋势白线 > 大哥黄线, 20天) AND
               EVERY(趋势白线 >= REF(趋势白线,1), 11天) AND
               红肥绿瘦

    // 超牛股
    超牛股 = (EVERY(BBI >= REF(BBI,1)*0.999, 20天) OR COUNT(BBI >= REF(BBI,1), 25天) >= 23) AND
            (近期振幅 >= 30 OR 远期振幅 > 80) AND
            BARSLAST(CROSS(C, 大哥黄线)) > 12

    RETURN 做上涨趋势, 强趋势股, 超牛股
```

## 7. 回踩判定

```
FUNCTION 回踩判定(C, L, O, 趋势白线, BBI, 大哥黄线, 当日涨跌幅):
    距离白线 = ABS(C - 趋势白线) / C * 100
    L距离白线 = ABS(L - 趋势白线) / 趋势白线 * 100
    距离BBI = ABS(C - BBI) / C * 100
    L距离BBI = ABS(L - BBI) / BBI * 100
    距离黄线 = ABS(C - 大哥黄线) / 大哥黄线 * 100

    // 回踩白线
    回踩白线 = (C >= 趋势白线 AND 距离白线 <= 2) OR
               (C < 趋势白线 AND 距离白线 < 0.8) OR
               (C >= BBI AND 距离BBI < 2.5 AND L距离BBI < 1 AND 距离白线 <= 3 AND 当日涨跌幅 < 1 AND C > REF(C,1))

    白线支撑 = (C >= 趋势白线 AND 距离白线 < 1.5)

    强势回踩不破 = (L距离白线 < 1 OR L距离BBI < 0.5) AND (C > 趋势白线) AND (距离白线 <= 3.5)

    // 回踩黄线
    回踩黄线 = (C >= 大哥黄线 AND (距离黄线 <= 1.5 OR (距离黄线 <= 2 AND 当日涨跌幅 < 1))) OR
               (C < 大哥黄线 AND 距离黄线 <= 0.8)

    RETURN 回踩白线, 白线支撑, 强势回踩不破, 回踩黄线, 距离白线, 距离BBI, 距离黄线
```

## 8. B1买入信号判定

```
FUNCTION B1买入信号(所有指标):
    // 信号1: 超卖缩量拐头B
    超卖缩量拐头B = 做上涨趋势 AND
                   (RSI - 15) >= REF(RSI,1) AND
                   (REF(RSI,1) < 20 OR REF(J,1) < 14) AND
                   当日振幅 < (振幅区间 + 0.5) AND
                   (当日涨跌幅 < 2.3 OR 上涨十字星) AND
                   (不是大绿棒 OR 大绿棒离得远) AND
                   (近期异动 OR 远期异动 OR 洗盘异动) AND
                   C >= 大哥黄线

    // 信号2: 超卖缩量B
    超卖缩量B = 做上涨趋势 AND
               (J < 14 OR RSI < 23) AND
               (RSI + J < 55 OR J = LLV(J,20)) AND
               当日振幅 < 振幅区间 AND
               (当日涨跌幅 < 2.5 OR 上涨十字星) AND
               (不是大绿棒 OR 大绿棒离得远) AND
               (缩量 OR (适当缩量 AND 当日涨跌幅 < 1)) AND
               (近期异动 OR 远期异动 OR 洗盘异动)

    // 信号3: 原始B1
    原始B1 = 趋势白线 > 大哥黄线 AND
            C >= 大哥黄线 * 0.99 AND
            大哥黄线 >= REF(大哥黄线, 1) AND
            (J < 13 OR RSI < 21) AND
            (RSI + J) < LLV(RSI+J, 15) * 1.5 AND
            适当缩量 AND
            (不是大绿棒 OR 大绿棒离得远) AND
            (ABS(C-O)*100/O < 1.5 OR 超缩量 OR (适当缩量 AND (距离白线 < 1.8 OR 距离BBI < 1.5 OR 距离黄线 < 2.8))) AND
            (近期异动 OR 远期异动 OR 洗盘异动)

    // 信号4: 超卖超缩量B
    超卖超缩量B = 做上涨趋势 AND
                (J < 14 OR RSI < 23) AND
                RSI + J < 60 AND
                远期振幅 >= 45 AND
                (当日振幅 < 振幅区间 OR (超级异动 AND 当日振幅 < 振幅区间+3.2 AND C > O AND C > 趋势白线)) AND
                ((C < O AND V < REF(V,1) AND C >= 大哥黄线) OR (C >= O)) AND
                (当日涨跌幅 < 2 OR 上涨十字星) AND
                (不是大绿棒 OR 大绿棒离得远) AND
                超缩量 AND
                (近期异动 OR 远期异动 OR 洗盘异动)

    // 信号5: 回踩白线B
    回踩白线B = 强趋势股 AND
               (J < 30 OR RSI < 40 OR 洗盘异动) AND
               RSI + J < 70 AND
               (当日振幅 < 振幅区间+0.5 OR 距离白线 < 1 OR 距离BBI < 1) AND
               回踩白线 AND
               (当日涨跌幅 < 2 OR (当日涨跌幅 < 5 AND 白线支撑)) AND
               (不是大绿棒 OR 大绿棒离得远) AND
               回踩缩量 AND
               (近期异动 OR 远期异动 OR 洗盘异动) AND
               L <= REF(C,1)

    // 信号6: 回踩超级B
    回踩超级B = 超牛股 AND
               (J < 35 OR RSI < 45 OR 洗盘异动) AND
               RSI + J < 80 AND
               (RSI + J) = LLV(RSI+J, 25) AND
               当日振幅 < 振幅区间 + 1 AND
               (当日涨跌幅 < 2.5 OR 距离白线 < 2) AND
               强势回踩不破 AND
               (不是大绿棒 OR 大绿棒离得远) AND
               (近期异动 OR 远期异动 OR 洗盘异动) AND
               适当缩量

    // 信号7: 回踩黄线B
    回踩黄线B = 趋势白线 >= 大哥黄线 AND
               C >= 大哥黄线 * 0.975 AND
               (J < 13 OR RSI < 18) AND
               回踩黄线 AND
               (不是大绿棒 OR 大绿棒离得远) AND
               (缩量 OR (适当缩量 AND (J = LLV(J,20) OR RSI = LLV(RSI,14)))) AND
               大哥黄线 >= REF(大哥黄线,1) * 0.997 AND
               MA(C,60) >= REF(MA(C,60),1) AND
               近期振幅 >= 11.9 AND
               远期振幅 >= 19.5

    RETURN 超卖缩量拐头B, 超卖缩量B, 原始B1, 超卖超缩量B, 回踩白线B, 回踩超级B, 回踩黄线B
```

## 9. 主程序 - 选股

```
FUNCTION 主程序_超级B1选股(股票代码, OHLCV数据):
    // Step 1: 计算基础指标
    趋势白线, 大哥黄线, 短期, 长期, BBI = 计算基础指标()

    // Step 2: 基础判定
    振幅区间, 当日振幅, 当日涨跌幅, 上涨十字星, 单针下20, 聚宝盆, 双叉戟, 红肥绿瘦 = 基础判定(股票代码, 历史数据)

    // Step 3: 成交量判定
    不是大绿棒, 大绿棒离得远, 缩量, 回踩缩量, 适当缩量, 超缩量 = 成交量判定(VOL, 历史成交量)

    // Step 4: 计算KDJ和RSI
    K, J, RSI = 计算KDJ_RSI(收盘价序列, 最高价序列, 最低价序列)

    // Step 5: 异动判定
    近期异动, 远期异动, 超级异动, 洗盘异动, 近期振幅, 远期振幅 = 异动判定(H, L, 20, 50)

    // Step 6: 趋势股判定
    做上涨趋势, 强趋势股, 超牛股 = 趋势股判定(趋势白线, 大哥黄线, C, O, BBI)

    // Step 7: 回踩判定
    回踩白线, 白线支撑, 强势回踩不破, 回踩黄线, 距离白线, 距离BBI, 距离黄线 = 回踩判定(C, L, O, 趋势白线, BBI, 大哥黄线, 当日涨跌幅)

    // Step 8: B1买入信号判定
    信号1, 信号2, 信号3, 信号4, 信号5, 信号6, 信号7 = B1买入信号(所有指标)

    // Step 9: 最终判定 - 满足任意一种信号即买入
    IF 信号1 OR 信号2 OR 信号3 OR 信号4 OR 信号5 OR 信号6 OR 信号7:
        RETURN TRUE  // 选中该股票
    ELSE:
        RETURN FALSE
```

## 10. 七种B1信号概览

| 信号类型 | 核心逻辑 | 趋势要求 |
|---------|---------|---------|
| 超卖缩量拐头B | RSI拐头向上，超卖后反弹 | 做上涨趋势 |
| 超卖缩量B | KDJ/RSI超卖，缩量 | 做上涨趋势 |
| 原始B1 | 经典B1形态，白线在黄线上方 | 白线>黄线 |
| 超卖超缩量B | 超卖+极度缩量，适合大振幅 | 做上涨趋势 |
| 回踩白线B | 强趋势股回踩白线不破 | 强趋势股 |
| 回踩超级B | 超牛股回踩，RSI+J创新低 | 超牛股 |
| 回踩黄线B | 回踩黄线，中期趋势向上 | 白线≥黄线 |
